## 🛠️ How to Use New Features

### **Running Tests:**
```bash
npm run test          # Interactive mode
npm run test:run      # CI mode
npm run test:ui       # Visual test runner
npm run test:coverage # Coverage report
```

### **Performance Monitoring:**
```bash
npm run lighthouse        # Full audit
npm run lighthouse:desktop # Desktop specific
npm run performance      # Build + audit
```

### **PWA Installation:**
- Visit the app in a supported browser
- Look for "Install App" prompt or menu option
- Add to home screen on mobile
- Use offline once installed

### **Error Monitoring:**
- Errors automatically logged to console
- Development mode shows full error details
- Production mode shows user-friendly messages
- Check browser localStorage for error history
