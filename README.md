# 🇰🇷 K-Learn Interactive - Korean Language Learning Platform

[![CI/CD Pipeline](https://github.com/k-learn-interactive/k-learn-interactive/workflows/CI/CD%20Pipeline/badge.svg)](https://github.com/k-learn-interactive/k-learn-interactive/actions)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A modern, interactive Korean language learning platform built with React, TypeScript, and AI-powered features. Master Korean through interactive lessons, cultural exploration, and AI conversations.

## ✨ Features

### 🎯 **Core Learning Modules**
- **Hangul Mastery** - Interactive Korean alphabet learning with pronunciation
- **Vocabulary Builder** - 200+ essential Korean words with categories
- **Grammar Patterns** - Fundamental Korean grammar with examples
- **Phrase Library** - Common Korean phrases for daily conversation
- **Cultural Insights** - Deep dive into Korean culture, regions, and modern society

### 🤖 **AI-Powered Learning**
- **Conversation Bot** - Practice Korean with Gemini AI
- **Adaptive Quizzes** - Smart testing with multiple question types
- **Personalized Progress** - Track your learning journey

### 🌍 **Cultural Exploration**
- **Regional Explorer** - Interactive map of South Korea with real satellite imagery
- **Daily Life Insights** - Learn about Korean lifestyle and customs
- **Modern Korea** - K-pop, technology, and contemporary society
- **Cultural Tips** - Authentic insights into Korean culture

### 🔍 **Advanced Features**
- **Spotlight Search** - Global search across all content (Ctrl/Cmd+K)
- **Progress Tracking** - Comprehensive learning analytics
- **Bookmark System** - Save and review favorite content
- **Dark Mode** - Eye-friendly learning environment
- **PWA Support** - Install as mobile/desktop app
- **Offline Access** - Learn even without internet

### 📱 **Modern UI/UX**
- **Fully Responsive** - Perfect on mobile, tablet, and desktop
- **Touch-Friendly** - Optimized for all input methods
- **Accessibility** - WCAG compliant with screen reader support
- **Performance Optimized** - Fast loading and smooth interactions

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/k-learn-interactive/k-learn-interactive.git
   cd k-learn-interactive
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   # Create .env.local file
   echo "GEMINI_API_KEY=your_gemini_api_key_here" > .env.local
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate to `http://localhost:5174`
